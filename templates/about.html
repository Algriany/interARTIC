About InterARTIC

InterARTIC allows users to run the ARTIC pipeline through a streamlined web application. Default parameters have been set from the existing protocols available.

These are the folowing protocols:

  1. Artic primers & Native BC
  2. Eden primers & Native BC
  3. Eden primers & Rapid BC
  4. Other

In addition, the user can modify these settings to their own preferences
