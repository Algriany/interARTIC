About InterARTIC

InterARTIC allows users to run the ARTIC pipeline through a streamlined web application.

Default parameters have been set from the existing protocols available:

SARS-CoV-2:
  1. Artic (400bp) - V1, V2, V3, V4
  2. Eden (2500bp) - V1
  3. Midnight (1200bp) - V1
IturiEBOV:
  1. Artic - V1
Custom:
  1. Input the path, virus, version of your own primer set

To have your primer set added to InterARTIC, please submit an issue on the github repo
